# AI-Based E-Learning Web Application

## Overview

An AI-powered e-learning platform built with Django that enables students to enroll in courses, watch learning videos, track their progress, complete AI-generated assessments, and download certificates upon successful completion.

## Features

- User authentication
- Course enrollment and management
- Video-based learning
- Progress tracking
- AI-generated assessments using Gemini API
- Automatic result evaluation
- Certificate generation
- Student dashboard
- Responsive user interface

# Tech Stack 

- Programming Language : Python 
- Backend framework : Django
- Frontend : HTML,CSS,Javascript
- Responsive UI : Bootstarp
- Database : SQLite
- AI Assessment Generation : Google Gemini API
